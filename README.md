# 嵌入式 Linux 智能家居中控系统

## 项目简介
这是一个适用于 GEC6818 等开发板的嵌入式 Linux 智能家居中控系统。系统以 LCD 触摸屏为主交互入口，包含环境监测、硬件控制、多媒体播放和离线语音识别功能。代码实现了具体的应用层逻辑，并在图形渲染、并发处理和传感器通信方面进行了底层调整。

## 技术栈与开发环境
* 开发语言使用 C 语言。
* 操作系统基于 Embedded Linux。
* 核心技术涉及 Linux 文件 I/O、进程控制、多路复用与非阻塞轮询、串口通信 (termios)、FrameBuffer 图形渲染和输入子系统 (Input Subsystem)。
* 第三方组件选用了 Madplay 用于音频解码，以及科大讯飞离线语音 SDK 用于 ASR。

## 核心功能模块

### 1. 智能环境感知与联动
系统通过 `/dev/ttySAC2` 和 `/dev/ttySAC3` 串口实时读取 GY39（光照、温度、湿度）与 MQ2（烟雾、可燃气体）传感器的数据。当光照低于 50 LUX 时，程序会自动开启补光 LED。若检测到有害气体浓度超过 500 PPM，界面会以红色高亮显示，并触发 PWM 蜂鸣器报警。

### 2. AI 离线语音交互
项目集成了科大讯飞离线语法识别 SDK，在无网环境下通过自定义 BNF 语法网络解析语音指令（例如“打开相册”）。语音监听与触控事件在主循环中并行处理。

### 3. 多媒体影音中控
音乐播放器基于 `madplay` 开发，通过 `system` 调用和进程信号控制播放、暂停及切歌操作。电子相册支持解析和显示 BMP 图像，用户可以通过触摸屏浏览本地图库。

### 4. 基础外设控制
用户界面提供触控面板，程序通过读写 Linux 设备节点（`/dev/led_drv` 和 `/dev/pwm`），实现对开发板载 LED 和蜂鸣器的手动控制。

## 技术实现细节

本项目在基础代码逻辑之外，处理了以下几个具体的硬件交互问题：

### 1. UI 渲染与内存映射
为解决读取大尺寸 BMP 图片缓慢和切屏卡顿的问题，代码使用 `mmap` 将图片文件直接映射到进程虚拟内存，替代标准的 `read()` 拷贝。内存中维护了 `RGB565` 格式的全局显存缓存，加快了界面切换速度。

### 2. 局部刷新机制
在传感器数据展示界面，系统计算数据变化的具体矩形区域。程序仅对数值变化所在的像素块进行定向覆写，避免了全屏清空重绘导致的 LCD 屏幕闪烁现象。

### 3. 非阻塞轮询并发处理
在无 RTOS 的裸机环境下，触摸读取逻辑使用了带有超时机制的 `touch_get_tap_timeout`。系统以 2000ms 为周期阻塞等待用户输入，超时后程序继续读取串口传感器数据。这种单线程轮询设计避免了引入多线程互斥锁。

### 4. 触控热区坐标映射
主界面由全屏底图构成，代码（`ui.c` 和 `main.c`）中通过坐标映射定义了不可见的点击热区。系统对退出按钮等边界热区进行了坐标外扩（`EXIT_HIT_PAD_X`），以兼容触摸屏的点击偏差。

### 5. 传感器迟滞控制逻辑
光照自动开灯的模块中加入了状态转变判断，仅在当前光照状态与设备开关状态不匹配时才操作驱动。这防止了开灯补光反向触发光照传感器导致的设备频繁开关现象，并允许后续配置双阈值区间。

## 项目结构

```text
.
├── src/
│   ├── core/
│   │   ├── main.c          # 系统主入口、主循环与界面调度
│   │   ├── ui.c            # UI 组件绘制与 Hitbox 碰撞检测
│   │   ├── lcd.c           # 帧缓冲 FrameBuffer 底层驱动
│   │   ├── touch.c         # 输入子系统解析
│   │   └── asr.c           # 离线语音识别核心引擎集成
│   └── modules/
│       ├── led_beep.c      # 硬件控制台模块
│       ├── sensor.c        # GY39 、MQ2 传感器采集与联动逻辑
│       ├── music.c         # Madplay 音乐播放控制器
│       └── photo.c         # 电子相册控制器
├── assets/
│   ├── images/             # UI 设计稿、图标与高品质 BMP 底图
│   └── music/              # MP3 音乐测试资源
├── bin/                    # 编译产物及语音识别依赖库
└── Makefile                # 项目自动化构建脚本
```
