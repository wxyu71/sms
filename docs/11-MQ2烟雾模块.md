# 11-MQ2烟雾模块

## 1.硬件接线

```
MQ2          开发板
G    -------- GND
T    -------- Rx
R    -------- Tx
5V   -------- 5V
```



## 2.发送指令给MQ2

![image-20260316081744928](11-MQ2烟雾模块.assets/image-20260316081744928.png)

```C
//打开MQ2模块连接的串口文件
int mq2_fd = open(...)
if(mq2_fd == -1)
{

}

//给MQ2模块发送指令，触发模块工作
char mq2_cmd[9] = {0xFF, 0x01, 0x86, 0x00, 0x00, 0x00, 0x00, 0x00, 0x79};
write(mq2_fd, mq2_cmd, 9);
```



## 3.获取模块数据

![image-20260316082043369](11-MQ2烟雾模块.assets/image-20260316082043369.png)

返回的浓度值9个字节，其中烟雾浓度 smog = byte2 << 8 | byte3;

```C
//读取烟雾浓度
char mq2_data[9] = {0};
read(mq2_fd, mq2_data, 9);
int smog = mq2_data[2] << 8 | mq2_data[3];
```



## 4.封装接口函数实现获取烟雾浓度值

```C
/*
	mq2_get_data: 烟雾模块获取数据
	@com_x: 烟雾模块连接的串口
	返回值:
		成功 返回实际获取到的烟雾浓度值
		失败 -1
*/
int mq2_get_data(char *com_x)
{
	//打开MQ2模块连接的串口文件
    int mq2_fd = open(com_x, O_RDWR);
    if(mq2_fd == -1)
    {

    }
    
    //初始化串口
    init_tty(mq2_fd);
    
    //给模块发送指令
    char mq2_cmd[9] = {0xFF, 0x01, 0x86, 0x00, 0x00, 0x00, 0x00, 0x00, 0x79};
    write(mq2_fd, mq2_cmd, 9);
    
    //读取模块数据
    char mq2_data[9] = {0};
    int smog;
    read(mq2_fd, mq2_data, 9);
    if(mq2_data[0] == 0xFF && mq2_data[1] == 0x86)
    {
        smog = mq2_data[2] << 8 | mq2_data[3];
    }
    
    //返回数据
    return smog;
}
```
